# UI/UX Rules

## 概要

UI・UX・画面の作成・編集・デザイン変更など、UIやUXに関わる作業の指示が来たときに、`UI_UX.md` を読み込んでルールに従った上で作業を行う。

## 手順

ユーザーからUI・UX関連の作業依頼が来た場合は、以下を実施する。

1. `/Users/kanekohiroki/Desktop/emoff/UI_UX.md` を Read ツールで読み込む。
2. 読み込んだルールに必ず従って実装・修正を行う。
3. UI_UX.md にルールが未定義の場合は、一般的なFlutterのベストプラクティスに従う。

## 対象となる依頼例

- 画面を作成して
- UIを修正して
- ダイアログを追加して
- ボタンを変更して
- レイアウトを調整して
- デザインを変更して
- 画面を編集して
- ウィジェットを作って
- エラーダイアログを表示して
- フィードバックを表示して
