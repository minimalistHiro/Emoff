# Legal Docs Check

## 参照ファイル

- `PRIVACY_POLICY.md`
- `TERMS_OF_SERVICE.md`
- `privacy_policy.md`
- `terms_of_service.md`
- `BUSINESS_MODEL.md`
- `APP_STORE_TEXTS.md`

## 手順

1. 対象が利用規約、プライバシー、AI同意、ストア文言のどれかを分類する。
2. Markdown原本とアセット用小文字ファイルの整合を確認する。
3. AI送信、原文保存、データ削除、課金に関する矛盾を探す。
4. 法的判断が必要な箇所は断定せず、確認事項として整理する。

## 注意事項

- 法律助言として断定しない。
- 実装やストア文言と矛盾する場合は修正候補を示す。
