# App Version Upgrade Rules

## 概要

emoffアプリのバージョン更新時に、pubspec.yaml のバージョンを確実に更新する。

## 手順

ユーザーから「バージョンをアップして」「バージョンを上げて」等の依頼が来た場合、以下を順番どおりに実施する。

1. `/Users/kanekohiroki/Desktop/emoff/pubspec.yaml` を Read ツールで読み込み、現在のバージョンを確認する。
2. `pubspec.yaml` のアプリバージョンを更新する。
   - パッチバージョン（`○.○.☆` の ☆）を1つ繰り上げる。
   - ビルド番号（`+○○`）を1つ上げる。
   - 例: `1.0.0+1` → `1.0.1+2`
3. アプリ内にバージョン表記が存在する場合は、同様に更新する（`lib/` 以下を Grep で検索して確認する）。
4. 更新内容を日本語で報告する。

## 注意事項

- ユーザーから特定のバージョン番号が指定された場合は、そのバージョンに更新する。
- ビルド番号は必ずインクリメントする。
