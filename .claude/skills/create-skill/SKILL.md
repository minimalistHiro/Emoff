# Create Skill

## 概要

ユーザーから新しいスキルの作成依頼が来たときに、`.claude/skills/` に SKILL.md を作成する。

## 手順

### 1. スキルの内容をユーザーに確認する

依頼が曖昧な場合は、以下を確認する：

- **スキル名**: kebab-case で指定（例：`my-new-skill`）
- **トリガー条件**: どのような依頼でこのスキルが発動するか
- **実行内容**: スキルが行う具体的な処理・手順

ユーザーがある程度の内容を伝えている場合は確認を最小限にし、すぐに作成に進む。

### 2. SKILL.md を作成する

以下のテンプレートに従って SKILL.md を作成する：

```markdown
---
name: <スキル名（kebab-case）>
description: <スキルの説明。トリガーとなる依頼例も含める。>
---

# <スキル表示名>

## 概要

（スキルの目的・背景を記載）

## 手順

（具体的な実行手順をステップで記載）

## 対象となる依頼例

（このスキルが発動する依頼の例を箇条書きで記載）

## 注意事項

（守るべきルール・制約を記載）
```

### 3. `.claude/skills/` に配置する

`/Users/kanekohiroki/Desktop/emoff/.claude/skills/<スキル名>/SKILL.md` に作成する。

### 4. 作成結果を報告する

作成したスキル名とパスを日本語で報告する。

## 対象となる依頼例

- スキルを作成して
- スキルを追加して
- 新しいスキルを作って
- 〇〇のスキルを作成して
- 〇〇用のスキルを追加して

## 注意事項

- スキル名は kebab-case（例：`my-new-skill`）を使用する。
- 既存スキルと名前が重複しないよう、作成前に既存スキル一覧を確認する（`ls /Users/kanekohiroki/Desktop/emoff/.claude/skills/`）。
