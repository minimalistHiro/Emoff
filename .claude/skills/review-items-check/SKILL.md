# Review Items Check

## 概要

要検討事項・未決定事項・検討中の事項に関する質問・確認・更新依頼が来たときに、`REVIEW_ITEMS.md` を読み込んで内容を確認・回答する。

## 手順

ユーザーから要検討事項に関する依頼が来た場合は、以下を実施する。

1. `/Users/kanekohiroki/Desktop/emoff/REVIEW_ITEMS.md` を Read ツールで読み込む。
2. 読み込んだ内容をもとに、ユーザーの質問・依頼に日本語で回答する。
3. 項目の追加・更新・状態変更を依頼された場合は、REVIEW_ITEMS.md を Edit ツールで更新する。
4. 決定済み・見送りになった項目は状態を `[x] 決定済み（日付）` または `[x] 却下（日付）` に更新し、理由を記載する。

## 対象となる依頼例

- 要検討事項を確認して
- 要検討事項を見せて
- 検討中の事項は何がある？
- 未決定事項を教えて
- 何を決める必要がある？
- 検討事項に追加して
- 要検討事項を整理して
- この件を要検討事項に追加して
- 決定した事項を反映して
