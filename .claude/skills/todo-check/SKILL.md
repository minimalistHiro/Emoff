# TODO Check

## 概要

TODO・実装すべき事項・未着手タスクに関する質問・確認・更新依頼が来たときに、`TODO.md` を読み込んで内容を確認・回答する。

## 手順

ユーザーからTODOに関する依頼が来た場合は、以下を実施する。

1. `/Users/kanekohiroki/Desktop/emoff/TODO.md` を Read ツールで読み込む。
2. 読み込んだ内容をもとに、ユーザーの質問・依頼に日本語で回答する。
3. 項目の追加・更新・状態変更を依頼された場合は、TODO.md を Edit ツールで更新する。
4. 完了済みの項目は COMPLETED.md へ移動し、完了日や備考を記載する。

## 対象となる依頼例

- TODOを確認して
- TODOを見せて
- 実装するべきことは何がある？
- 未着手のタスクを教えて
- 何を実装する必要がある？
- 次に何をすればいい？
- TODOに追加して
- タスクの状況を教えて
- このタスクを完了にして
- TODOを整理して
- 残っているタスクは？
