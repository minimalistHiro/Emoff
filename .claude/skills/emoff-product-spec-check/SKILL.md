# Emoff Product Spec Check

## 参照ファイル

- `BUSINESS_MODEL.md`
- `APP_SCREENS.md`
- `SCREEN_DESIGN.md`
- `FIRESTORE.md`
- `TODO.md`
- `COMPLETED.md`
- `REVIEW_ITEMS.md`

## 手順

1. 質問の対象を機能、画面、データ、課金、運用に分類する。
2. 関連ファイルを読み、確定仕様と未決定事項を分けて回答する。
3. 仕様が未定なら `REVIEW_ITEMS.md` の該当項目または追加候補を示す。

## 注意事項

- 推測で仕様を断定しない。
- AI変換・原文保存・同意に関する内容は `ai-conversion-rules-check` と `legal-docs-check` も見る。
