# ログイン画面 — Welcome Back

- **作成日**: 2026-04-03
- **ステータス**: 設計済み・未実装
- **対象画面**: ログイン画面

---

## 画面概要

メールアドレスとパスワードでログインする画面。
「Welcome Back — Access your workspace for clarity.」というコピーで、Emoffの哲学（感情を排した明晰なコミュニケーション）を訴求する。

## レイアウト構成

**モバイル**: フォームカードのみ（中央配置、縦スクロール不要）
**デスクトップ（タブレット以上）**: 左右2カラム（左: ブランディングビジュアル、右: ログインフォーム）

※ Flutterモバイルアプリとしてはモバイルレイアウトのみ実装

## 画面構成（上から順 — モバイルレイアウト）

### 1. ブランディング領域

| 要素 | 詳細 |
|------|------|
| アプリ名「EMOFF」 | 大文字・ボールド・レタースペーシング広め（tracking-widest）、20px |
| 配置 | 中央寄せ |

- **テキスト色**: `#00D4FF`（シアン）
- **下マージン**: 48px

### 2. ウェルカムヘッダー

| 要素 | 詳細 |
|------|------|
| メインタイトル | 「Welcome Back」— ボールド、24px、Manrope系ヘッドラインフォント |
| サブテキスト | 「Access your workspace for clarity.」— 通常ウェイト、14px |

- **配置**: 左寄せ
- **タイトル色**: `#FFFFFF`
- **サブテキスト色**: `#A0A0A0`
- **下マージン**: 40px

### 3. メールアドレス入力フィールド

| 要素 | 詳細 |
|------|------|
| ラベル | 「EMAIL ADDRESS」— 大文字、11px、レタースペーシング広め、`#A0A0A0` |
| 入力フィールド | プレースホルダー「name@domain.com」 |
| キーボードタイプ | `emailAddress` |

- **実装時**: `CustomTextField` を使用
- **背景色**: `#1A1A1A`
- **角丸**: 12〜16px（xl相当）
- **下マージン**: 24px

### 4. パスワード入力フィールド

| 要素 | 詳細 |
|------|------|
| ラベル行（左） | 「PASSWORD」— 大文字、11px、レタースペーシング広め、`#A0A0A0` |
| ラベル行（右） | 「FORGOT PASSWORD?」— 大文字、11px、`#00D4FF`（シアン）、タップでパスワードリセット画面へ |
| 入力フィールド | プレースホルダー「••••••••」、パスワード非表示（obscureText: true） |

- **実装時**: `CustomTextField` を使用
- **ラベル行**: `Row` で左右に `SpaceBetween` 配置
- **下マージン**: 16px

### 5. ログインボタン

| 要素 | 詳細 |
|------|------|
| テキスト | 「Login」+ 右矢印アイコン（`arrow_forward`） |

- **上マージン**: 16px
- **タップ時**: `scale(0.98)` のフィードバックアニメーション
- **影**: 軽いドロップシャドウ

### 6. 区切り線 + 新規登録リンク

| 要素 | 詳細 |
|------|------|
| 区切り線 | 水平線、`#242424`、1px |
| テキスト | 「New to the philosophy?」— 14px、`#A0A0A0` |
| リンク | 「Sign Up」— 14px、セミボールド、`#00D4FF`（シアン） |

- **配置**: 中央寄せ
- **上マージン**: 48px（区切り線まで）+ 32px（テキストまで）
- **タップ動作**: 新規登録画面へ遷移

## 背景演出

| 要素 | 詳細 |
|------|------|
| メイン背景 | `#0D0D0D`（ダーク） |
| フォームカード背景 | `#1A1A1A`（やや明るいダーク）または背景と同色 |
| アンビエントグロー（任意） | 画面右上と左下に、シアン/パープルの大きなぼかし円（blur: 120px、opacity: 5%）を配置し、奥行き感を演出 |

## インタラクション

| アクション | 動作 |
|------------|------|
| ログインボタンをタップ | Firebase Auth でメール+パスワード認証 → 成功時ホーム画面へ遷移 |
| 「FORGOT PASSWORD?」をタップ | パスワードリセット画面へ遷移 |
| 「Sign Up」をタップ | 新規登録画面へ遷移 |
| バリデーションエラー | `CustomDialog` でエラーメッセージ表示（スナックバー禁止） |
| 認証エラー | `toUserFriendlyError()` で変換した日本語メッセージを `CustomDialog` で表示 |

## デザインノート

- **ミニマル構成**: フォーム要素のみの潔い画面。不要な装飾を排除し、Emoffの「感情を排す」哲学を体現
- **ラベルスタイル**: すべて大文字+広めのレタースペーシング（tracking-widest）で、エディトリアル/ハイエンドな印象
- **フォームカード**: モバイルでは画面中央に浮かぶカード風のレイアウト（角丸大・軽い影）
- **デスクトップ左カラム（参考）**: 「Objective Communication.」のキャッチコピーとミニマルな建築写真を配置。モバイルアプリでは不要だが、Web版展開時の参考として保持
