# お知らせ画面 — Announcements

- **作成日**: 2026-04-03
- **ステータス**: 設計済み・未実装
- **対象画面**: お知らせ画面

---

## 画面概要

アプリからのアップデート情報・システム通知を時系列で一覧表示する画面。
設定画面の「Announcements」から遷移する。
管理者がFirebaseコンソールから投稿した `announcements` コレクションのデータを表示する。

---

## 画面構成（上から順）

### 1. AppBar（固定ヘッダー）

| 要素 | 位置 | 詳細 |
|------|------|------|
| 戻るボタン | 左端 | `arrow_back` アイコン |
| 画面タイトル「Announcements」 | 左寄せ（アイコン右） | ボールド、20px、Manrope |

- **実装時**: `CustomAppBar` を使用
- **背景色**: `#0D0D0D`
- **タイトル色**: `#FFFFFF`

### 2. お知らせリスト

`announcements` コレクションを `createdAt` 降順で取得し、カード形式で一覧表示する。

#### お知らせカード

| 要素 | 詳細 |
|------|------|
| タイトル | `announcements.title` — ボールド、16px、`#FFFFFF` |
| 日付 | `announcements.createdAt` — 12px、`#A0A0A0`、「2026年4月3日」形式 |
| 本文 | `announcements.body` — 通常ウェイト、14px、`#A0A0A0`、最大3行で `overflow: ellipsis` |
| 展開状態の本文 | 全文表示 — 通常ウェイト、14px、`#A0A0A0` |

**カードスタイル:**
- 背景色: `#1A1A1A`
- 角丸: 12px
- パディング: 16px
- 左右マージン: 16px
- カード間マージン: 12px
- タイトルと日付の間隔: 4px
- 日付と本文の間隔: 12px

**展開/折りたたみ:**
- 本文が3行を超える場合、カード下部に「もっと見る」テキスト表示（`#00D4FF`、12px）
- タップでカード内の本文が全文展開。「閉じる」テキストに切り替わる
- アニメーション: `AnimatedCrossFade` または `AnimatedSize` でスムーズに展開

### 3. 空状態

お知らせが0件の場合に表示する。

| 要素 | 詳細 |
|------|------|
| アイコン | `notifications_none`、64px、`#555555`、中央配置 |
| テキスト | 「お知らせはありません」— 16px、`#555555`、中央配置 |

- **配置**: 画面中央（`Center` ウィジェット）

### 4. ローディング状態

初回データ取得中に表示。

| 要素 | 詳細 |
|------|------|
| インジケーター | `CustomLoadingIndicator`、画面中央 |

## Firestoreとの対応

| UI要素 | Firestoreクエリ/フィールド |
|--------|--------------------------|
| お知らせリスト | `announcements` コレクション、`orderBy('createdAt', 'desc')` |
| タイトル | `announcements/{id}.title` |
| 日付 | `announcements/{id}.createdAt` |
| 本文 | `announcements/{id}.body` |

- リアルタイムリスナー（`snapshots()`）ではなくワンショット取得（`get()`）で十分。頻繁に更新されるデータではないため
- ページネーション: 当面は全件取得。件数が増えた場合は `limit(20)` + 「もっと読み込む」ボタンで対応

## インタラクション

| アクション | 動作 |
|------------|------|
| 戻るボタンをタップ | 設定画面に戻る |
| お知らせカードの「もっと見る」をタップ | カード内で本文を全文展開 |
| 展開状態で「閉じる」をタップ | 本文を3行に折りたたむ |
| リストを下に引く | プルトゥリフレッシュでデータ再取得 |

---

## エッジケース

| ケース | 対応 |
|--------|------|
| お知らせが0件 | 空状態（アイコン + テキスト）を表示 |
| 本文が3行以内 | 「もっと見る」を表示しない |
| ネットワークエラー | エラーダイアログ:「通信エラーが発生しました」。リトライはプルトゥリフレッシュで |
| 本文に長いURLが含まれる | テキストの折り返し処理。URLは自動リンク化しない（シンプルなテキスト表示） |

---

## デザインノート

- **カード形式**: 各お知らせを独立したカード（`#1A1A1A` 背景）で表示し、情報の区切りを明確に
- **展開/折りたたみ**: 詳細画面への遷移ではなく、カード内で本文を展開するインライン方式を採用。画面遷移のオーバーヘッドを削減し、複数のお知らせを効率的に閲覧可能
- **ワンショット取得**: お知らせは管理者が不定期に投稿するものであり、リアルタイム性は不要。プルトゥリフレッシュで手動更新
- **日付フォーマット**: 「2026年4月3日」の日本語形式。時刻は表示しない（粒度が細かすぎるため）
- **アイテムにアイコンなし**: 設定画面群と統一し、ミニマルなテキスト中心の構成
