# トーク一覧画面 — Communication Hub

- **作成日**: 2026-04-03
- **ステータス**: 設計済み・未実装
- **対象画面**: トーク一覧画面

---

## 画面概要

過去のトーク履歴を新着順に一覧表示する画面。個別チャットとグループチャットの両方を統合的に表示する。
各トークアイテムにはAI処理状態を示すバッジ（NEUTRALIZED / VERIFIED）が付与され、Emoffのトーン管理が一目で把握できる。

## 画面構成（上から順）

### 1. AppBar（固定ヘッダー）

| 要素 | 位置 | 詳細 |
|------|------|------|
| ハンバーガーメニュー | 左端 | `menu` アイコン。サイドドロワーを開く |
| アプリ名「EMOFF」 | 左寄せ（アイコン右） | 大文字・ボールド・レタースペーシング広め（tracking-widest）、18px |
| 検索アイコン | 右側① | `search` アイコン。トーク検索 |
| プロフィールアバター | 右側② | 32×32px、円形、自分のプロフィール画像。タップでプロフィール/設定への導線 |

- **実装時**: `CustomAppBar` を使用
- **背景色**: `#0D0D0D`
- **下部**: 1pxの区切り線（`#242424`）

### 2. セクションヘッダー

| 要素 | 詳細 |
|------|------|
| サブラベル | 「COMMUNICATION HUB」— 大文字、12px、レタースペーシング広め、`#A0A0A0` |
| メインタイトル | 「Chats」— 極太（ExtraBold）、30px、Manrope |
| 「New Discussion」ボタン | 右寄せ。カプセル型、`#00D4FF`背景、`#0D0D0D`テキスト、14px、ミディアムウェイト |

- **配置**: サブラベル+タイトルは左寄せ、ボタンは右寄せ（`SpaceBetween`）
- **タイトル色**: `#FFFFFF`
- **上下パディング**: 32px

### 3. トークリスト

各トークは横並びのカード形式で表示。

#### トークアイテム1件あたりの構成

| 要素 | 詳細 |
|------|------|
| アバター画像 | 56×56px、円形。個別チャットはユーザー写真（軽めのグレースケール `grayscale-[20%]`）、グループはアイコン（`architecture`等のMaterial Icon、背景付き円） |
| オンラインステータス | アバター右下に小さな丸（直径14px）。オンライン=`#00D4FF`（シアン）。個別チャットのみ表示 |
| 名前 | ボールド、16px、`#FFFFFF` |
| 日時 | 名前の右横、12px、`#A0A0A0`。「10:42 AM」「Yesterday」「Tuesday」「Monday」等 |
| 最新メッセージプレビュー | 1行、15px、省略表示（truncate）。アクティブ: `#A0A0A0`の濃いめ + ミディアムウェイト、非アクティブ: `#A0A0A0` |
| AI処理バッジ | メッセージプレビューの右横（後述） |
| 未読インジケーター | 右端に小さな丸（直径8px、`#00D4FF`）。未読メッセージがある場合のみ表示 |
| ホバーメニュー | `more_horiz` アイコン。ホバー時のみ表示（opacity: 0 → 1） |

**カードスタイル:**
- アクティブ（最新/選択中）: 背景 `#1A1A1A`（surface-container-low）
- 非アクティブ: 背景透明、ホバー時 `#1A1A1A`
- 角丸: 12〜16px
- パディング: 20px
- アイテム間スペース: 4px（space-y-1）

#### AI処理バッジの種類

| バッジ | スタイル | 意味 |
|--------|----------|------|
| **NEUTRALIZED** | 背景 `#5e5c78`（tertiary）、テキスト `#FFFFFF`、大文字、10px、ボールド | AI変換済み（トーン中立化が適用されたメッセージ） |
| **VERIFIED** | 背景 `#5e5c78` の10%不透明度、テキスト `#5e5c78`、大文字、10px、ボールド | AI検証済み（元からトーンに問題なしと判定されたメッセージ） |
| （バッジなし） | — | 受信メッセージ等、AI処理対象外 |

#### グループチャットアイテムの違い

| 要素 | 個別チャットとの差異 |
|------|---------------------|
| アバター | ユーザー写真ではなく、Material Iconを中央配置した円形背景（`#242424`背景、アイコン色 `#00D4FF`） |
| オンラインステータス | 非表示 |
| 名前 | グループ名（長い場合はtruncate） |

### 4. AI Concierge ステータスセクション

トークリスト下部に表示される、AI処理状態を伝えるエリア。

| 要素 | 詳細 |
|------|------|
| 枠線 | 破線ボーダー（dashed）、角丸 16px |
| アイコン | `auto_awesome`（スパークル）、36px、`#555555` |
| タイトル | 「AI Concierge Ready」— ボールド、18px、`#FFFFFF` |
| 説明テキスト | 「All messages are currently being processed to maintain a professional and objective tone.」— 14px、`#A0A0A0`、中央寄せ |

- **配置**: 中央寄せ、パディング 32px
- **上マージン**: 48px
- **背景**: 透明

### 5. ボトムナビゲーション（共通コンポーネント）

| タブ | アイコン | ラベル | 状態 |
|------|----------|--------|------|
| 友達 | `group` | FRIENDS | 非アクティブ |
| チャット | `chat_bubble` | CHATS | **アクティブ**（アイコンFILL=1、背景ハイライト） |
| 設定 | `settings` | SETTINGS | 非アクティブ |

（スタイルはホーム画面のボトムナビと同一）

## インタラクション

| アクション | 動作 |
|------------|------|
| トークアイテムをタップ | 該当のトークルーム画面へ遷移 |
| 「New Discussion」ボタンをタップ | 新規トーク作成（友達選択 or グループ作成） |
| 検索アイコンをタップ | トーク検索モード |
| プロフィールアバターをタップ | プロフィール設定画面 or 設定画面へ遷移 |
| ホバーメニュー（`more_horiz`）をタップ | アーカイブ・ミュート・削除等のオプション |
| ハンバーガーメニューをタップ | サイドドロワーを開く |

## デザインノート

- **バッジシステム**: 「NEUTRALIZED」と「VERIFIED」の2種類のバッジで、AI処理の種類を一覧上で即座に識別可能。NEUTRALIZEDは塗り背景（目立つ）、VERIFIEDは半透明背景（控えめ）で視覚的に差別化
- **グレースケール写真**: ホーム画面と同様、アバターは軽めのグレースケール（20%）を適用。完全なグレースケールではなく少し色味を残すことで、一覧での識別性を確保
- **グループアイコン**: グループチャットはMaterial Iconベースのアバターで、個別チャットと視覚的に区別
- **AI Conciergeセクション**: EmoffのAI機能が常にアクティブであることをユーザーに安心感として伝えるUI要素
- **未読インジケーター**: 小さなドット（8px）で最小限の主張。数字カウントは使用せず、ミニマルな表現
