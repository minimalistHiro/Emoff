- **作成日**: 2026-04-03

- **対象画面**: プロフィール設定画面

---

## 画面概要

自分のプロフィール（背景画像・アイコン・名前・ユーザーID・自己紹介文）を編集する画面。
設定画面の「Edit Profile」ボタンから遷移する。
友達プロフィール画面（閲覧専用）と視覚的に対応しつつ、各要素が編集可能になっている。
自己紹介文はチャットと同様にAI変換を通した文章のみ保存される。

## 保存処理フロー

### 通常フィールド（名前・ユーザーID）の保存

```
[Saveボタンタップ]
  ├── バリデーション実行
  │     ├── 名前: 空欄チェック
  │     ├── ユーザーID: 空欄・3文字以上・一意性チェック
  │     └── エラーあり → 該当フィールドにエラー表示、保存中断
  ├── バリデーション通過
  │     ├── ローディング表示（保存ボタンを非活性化 + CustomLoadingIndicator）
  │     ├── name, userId を Firestore に更新
  │     └── 画像変更がある場合は Firebase Storage にアップロード → URL取得 → Firestore 更新
  ├── 成功 → 完了ダイアログ:「プロフィールを更新しました」→ 設定画面に戻る
  └── 失敗 → エラーダイアログ（toUserFriendlyError で変換）
```

### 自己紹介文のAI変換フロー

自己紹介文が変更されている場合、保存時にチャットと同様のAI変換プレビューを挟む。

```
[Saveボタンタップ（bioが変更されている場合）]
  ├── バリデーション通過
  ├── ローディング表示
  ├── AI変換API呼び出し（bio テキストを送信）
  ├── 変換結果をプレビューダイアログで表示（後述）
  │     ├── [確認] → 変換後テキストで bio を保存 → 完了
  │     └── [やり直す] → ダイアログを閉じて編集画面に戻る
  └── API失敗 → エラーダイアログ:「文章の変換に失敗しました。もう一度お試しください」
```

### AI変換プレビューダイアログ

| 要素 | 詳細 |
|------|------|
| タイトル | 「自己紹介の変換プレビュー」— ボールド、16px、`#FFFFFF` |
| 変換前ラベル | 「変換前」— 12px、ボールド、`#A0A0A0` |
| 変換前テキスト | 入力した原文 — 14px、`#A0A0A0`、`#1A1A1A` 背景、角丸 8px、パディング 12px |
| 矢印アイコン | `arrow_downward`、24px、`#00D4FF`、中央、上下マージン 12px |
| 変換後ラベル | 「変換後」— 12px、ボールド、`#00D4FF` |
| 変換後テキスト | AI変換結果 — 14px、`#FFFFFF`、`#1A1A1A` 背景、角丸 8px、パディング 12px、左ボーダー 2px `#00D4FF` |
| 「やり直す」ボタン | `CustomButton`（secondary バリアント） |
| 「確認」ボタン | `CustomButton`（primary バリアント） |

- **実装時**: `CustomDialog` を使用
- **背景色**: `#242424`
- **角丸**: 16px
- **ボタン配置**: 横並び（左: やり直す、右: 確認）、ボタン間隔 12px

## Firestoreとの対応

| UI要素 | Firestoreフィールド | 操作 |
|--------|-------------------|------|
| 背景画像 | `users/{myUid}.backgroundUrl` | 読取 + 更新（Firebase Storage URL） |
| アイコン画像 | `users/{myUid}.iconUrl` | 読取 + 更新（Firebase Storage URL） |
| 名前 | `users/{myUid}.name` | 読取 + 更新 |
| ユーザーID | `users/{myUid}.userId` | 読取 + 更新（一意性チェック付き） |
| 自己紹介文 | `users/{myUid}.bio` | 読取 + 更新（AI変換後テキスト） |
| 更新日時 | `users/{myUid}.updatedAt` | 更新（保存時に `serverTimestamp()` ） |

### 画像のFirebase Storageパス

| 画像 | パス |
|------|------|
| アイコン画像 | `users/{myUid}/icon.jpg` |
| 背景画像 | `users/{myUid}/background.jpg` |

- アップロード時に既存ファイルを上書き（同一パス）
- 削除時は Storage のファイルを削除し、Firestore の URL フィールドを `null` に更新

### 非正規化データの同期

プロフィール更新時、Cloud Functions が以下の非正規化データを自動同期する:
- `users/*/friends/{myUid}` の `name`, `iconUrl` を更新
- `friend_requests`（`fromUid == myUid`）の `fromName`, `fromIconUrl` を更新

| アクション | 動作 |
|------------|------|
| 戻るボタンをタップ | 変更がある場合は破棄確認ダイアログ表示。なければ設定画面に戻る |
| 背景画像エリアをタップ | 画像選択ボトムシート表示（背景画像用） |

| ユーザーIDフィールドを編集 | 自動小文字変換 + リアルタイム文字数カウンター更新 |
| 自己紹介文を編集 | リアルタイムで文字数カウンター更新 |
| Saveボタンをタップ | バリデーション → (bioが変更されていればAI変換プレビュー) → 保存処理 → 設定画面に戻る |

| 要素 | 詳細 |
|------|------|
| タイトル | 「変更を破棄しますか？」— ボールド、16px、`#FFFFFF` |
| 説明文 | 「編集中の内容は保存されません」— 14px、`#A0A0A0` |
| 「キャンセル」ボタン | `CustomButton`（secondary バリアント）→ ダイアログを閉じて編集を続行 |
| 「破棄」ボタン | `CustomButton`（danger バリアント）→ 変更を破棄して設定画面に戻る |

- **実装時**: `CustomDialog` を使用

## エッジケース

| ケース | 対応 |
|--------|------|
| 背景画像未設定 | グラデーション背景（`#1A1A1A` → `#242424`）+ カメラアイコン表示 |
| アイコン未設定 | デフォルトアバター（`person` アイコン）+ カメラバッジ表示 |
| 自己紹介文が空 | AI変換処理をスキップし、`bio` フィールドを空文字で保存 |
| 自己紹介文のみ変更 | AI変換プレビューダイアログを表示してから保存 |
| 画像のみ変更（テキストフィールド変更なし） | AI変換なしで画像のみアップロード・保存 |
| userId重複 | フィールド下にインラインエラー表示。Saveボタン非活性化 |
| 画像アップロード失敗 | エラーダイアログ:「画像のアップロードに失敗しました」。テキストフィールドの変更は保存しない（アトミック） |
| AI変換API失敗 | エラーダイアログ:「文章の変換に失敗しました。もう一度お試しください」 |
| カメラ/ライブラリ権限なし | 端末の設定画面への誘導ダイアログ表示 |
| ネットワークエラー | エラーダイアログ:「通信エラーが発生しました」 |
| 名前が空のまま保存 | フィールド下にインラインエラー。Saveボタン非活性化 |
| userIdが3文字未満 | フィールド下にインラインエラー:「3文字以上入力してください」 |
| 変更なしでSaveタップ | ボタン自体が非活性のためタップ不可 |

## デザインノート

- **友達プロフィール画面との一貫性**: 背景画像（200px）+ 重なるアイコン（96px）の構成は友達プロフィール画面と同じ。閲覧画面と編集画面で視覚的連続性を保つ
- **編集可能の視覚的ヒント**: 背景画像にはオーバーレイ + カメラアイコン、アイコンにはカメラバッジを配置し、タップして変更できることを直感的に伝える
- **AI変換プレビュー**: チャットの送信フロー（入力→確認→送信）と同じ思想。自己紹介文も「AIが整えた文章のみ公開される」というEmoffのコンセプトを一貫して適用
- **破棄確認ダイアログ**: 入力途中の内容を誤って消してしまうことを防ぐ安全策。dangerバリアントの「破棄」ボタンで破壊的アクションであることを明示
- **保存のアトミック性**: 画像アップロードとテキスト更新は全てが成功した場合のみ反映。部分的な更新による不整合を防ぐ
- **ユーザーIDの制約**: 半角英数字・アンダースコアのみ、自動小文字変換で統一。友達申請のID検索で利用するため、分かりやすく入力しやすい形式に限定
